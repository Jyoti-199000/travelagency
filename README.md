# TravelVista — Premium Travel Agency Website

A fully functional, responsive travel agency website inspired by MakeMyTrip UI/UX but with a cleaner, more professional design.

## 🌍 Live Features

- **5 Complete Pages**: Home, Destinations, Packages, About Us, Contact
- **Dynamic Search & Filtering**: Real-time search across destinations and packages
- **Booking Modal**: Interactive booking preview with price calculation
- **Hero Slider**: Auto-rotating hero carousel with stunning travel imagery
- **Testimonials Slider**: Customer reviews with navigation controls
- **Contact Form**: Full validation with success feedback
- **Google Maps**: Embedded office location
- **Responsive Design**: Mobile, tablet, and desktop optimized
- **Smooth Animations**: Scroll reveal, hover effects, loading skeletons

## 🛠 Tech Stack

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, Flexbox, Grid, animations
- **Vanilla JavaScript** — No frameworks or dependencies
- **Font Awesome 6** — Icon library
- **Google Fonts** — Inter + Playfair Display
- **Unsplash** — Dynamic travel imagery

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in any modern browser
3. No build tools or server required!

## 🌐 Deployment

This is a static site — deploy directly to:

- **Netlify**: Drag & drop the folder
- **Vercel**: Import the folder
- **GitHub Pages**: Push to a repo and enable Pages

## 📱 Responsive Breakpoints

- Desktop: 1024px+
- Tablet: 768px – 1024px
- Mobile: < 768px
- Small Mobile: < 480px
